## üß© What I Learned

### 1Ô∏è‚É£ Univariate Analysis
I learned that **univariate analysis** is all about **looking at one variable at a time**.  
It helps me **understand distribution, central values, and detect outliers**.  

> **Types of Data I worked with:**
> - *Numerical:* age, height, weight, price, etc.  
> - *Categorical:* gender, country, category, etc.  

> **Visualizations I tried:**
> - üìä **Bar Plot / Count Plot:** Count of categories  
> - ü•ß **Pie Chart:** Category proportions  
> - üìà **Histogram:** Frequency distribution for numbers  
> - üåä **Distribution Plot (KDE/Distplot):** Density visualization  
> - üì¶ **Box Plot:** Quick view of min, max, median, quartiles, and outliers  

> **Stats I calculated:**  
> Minimum, Maximum, Mean, Skewness ‚Äî helped me understand the data better.

### 2Ô∏è‚É£ Bivariate Analysis
I learned that **bivariate analysis** explores **two variables together** to understand relationships and patterns.  

> **Numerical vs Numerical:**
> - üîπ **Scatter Plot:** Check correlation  
> - üìâ **Line Plot:** Useful for trends or time-series data  

> **Categorical vs Numerical:**
> - üìä **Bar Plot:** Compare numerical values across categories  
> - üì¶ **Box Plot:** Compare distributions  
> - üåä **KDE Plot:** Compare density distributions  

> **Categorical vs Categorical:**
> - üó∫Ô∏è **Heatmap:** Check relationships between categories  
> - üìë **GroupBy:** Summarize data by category  

> ‚ö° *Tip:* Bivariate analysis is great for spotting **trends and patterns** that help in feature selection for ML projects.

---

### 3Ô∏è‚É£ Multivariate Analysis
I learned that **multivariate analysis** looks at **three or more variables together**.  

> **Techniques I used:**
> - **Enhanced Scatter Plots:** Add *hue*, *size*, or *style* to show extra info  
> - **Multivariate Bar Plots:** Compare a third category using *hue*  
> - **Pair Plots:** Visualize relationships among multiple numerical variables  
> - **Clustermap & Pivot Tables:** Reveal trends and patterns across multiple features  

> ‚ö° *Tip:* Multivariate analysis helps me **discover complex interactions** between variables.
